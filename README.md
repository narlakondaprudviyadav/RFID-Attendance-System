## 📌 Description
This project automates attendance using RFID technology. Each user has a unique RFID card. When scanned, the system verifies and marks attendance.

## ⚙️ Components Used
- Arduino UNO
- RC522 RFID Module
- RFID Cards
- LEDs (Red & Green)
- Buzzer

## 🔄 Working
1. RFID card is scanned
2. Arduino reads UID
3. If UID matches:
   - Green LED ON
   - Buzzer sound
4. Else:
   - Red LED ON
